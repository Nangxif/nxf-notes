什么的是SQL #

  • SQL是一门数据库编程语言
  • 使用SQL语言编写出来的代码叫做SQL语句
  • SQL语言只能在关系型数据库中使用（例如MySQL、Oracle、SQLServer）。非关系型数据库（例如MongoDB）不支持SQL语言
